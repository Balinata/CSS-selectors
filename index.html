Які бувають селектори CSS

Селектор Приклад Приклад опису
.class .intro Вибір всіх елементів із class="intro"
#id #firstname Вибір елемента з id="firstname"
[attribute^=value] a[href^="https"] Вибирає кожен елемент a, значення атрибуту href якого починається з "https"
:active a:active Вибір активного посилання
:first-child p:first-child Вибирає кожен елемент p,який є першим дочірнім елементом батьківського елемента
:enabled input:enabled Вибір всіх включених елементів input
:link a:link Вибір усіх посилань, що не відвідуються.
:target #news:target Вибір поточного активного елемента #news
:only-child p:only-child Вибирає кожен елемент p, який є єдиним дочірнім елементом батьківського елемента
:first-of-type p:first-of-type Вибір кожного елемента p, який є першим елементом батьківського елемента p
